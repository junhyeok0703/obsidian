또다른 자료형으로 사전이 있다. (딕셔너리)
- key-value pair(키-값 쌍)
```
my_dictionary = {
	5 : 25,
	2 : 4,
	3 : 9
}
```
키 : 값으로 구분 , 쉼표로 서로 구분
type은 dict임

접근하려면
```
my_dictionary[3]
-> 9 출력
# 삽입
my_dictionary[9] = 81
print(my_dictionary)
-> 추가된것을 확인가능
```

인덱스가 0,1,2,3,4이렇게 됬지만 키값을 아무순서로 할수있다
순서 x
