
### 리스트 복습하기
#### 리스트는 \[]를 사용한다.
```
a = [1,5,7]
print(a)
```
#### 다양한 자료형을 섞어서 쓸수있다.

```
a = [1,"안녕",4.2]
print(a)
```
#### 형변환
```
긴문자열을 리스트로 하나씩 잘라서 만들어줌
text_to_list = list("안녕하세요")
print(text_to_list)
```

#### 인덱싱
```
리스트의 값을 접근할수있음
a = [1,3,4,5]
print(a[1])
--> 3
```
#### 슬라이싱 
```
범위를 짤라서 가져올수있음
a = [1,3,4,5]
a[:3] -> 인덱스0~2까지 
a[2:] -> 인덱스2~마지막까지 
a[1:2] -> 인덱스1~1까지 
a[1:5] -> 인덱스1~마지막까지
뒤에 있는 숫자는 -1을 해야한다.
```
