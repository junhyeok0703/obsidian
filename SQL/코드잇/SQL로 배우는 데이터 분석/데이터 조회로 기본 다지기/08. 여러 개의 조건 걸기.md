
### 1. 25~29세이고 서울에 살고있는 남자 조회
```
-- 서울에 살고있는 남자 조회
SELECT * FROM member
WHERE gender = 'm' 
	AND address LIKE "서울%"
	AND age BETWEEN 25 AND 29;
```
### 2. OR사용하기 (봄이나 가을에 가입한 인원조회)
```
-- 둘중에 하나만 조건 맞아도 조회 봄또는 가을에 가입한 인원조회
SELECT * FROM member
WHERE MONTH(sign_up_day) BETWEEN 3 AND 5
	OR MONTH(sign_up_day) BETWEEN 9 AND 11;
```
