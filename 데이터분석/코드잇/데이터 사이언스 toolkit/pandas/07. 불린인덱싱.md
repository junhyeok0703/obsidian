
### 조건을 걸면 TRUE , FALSE로 알려줌

100보다큰거 = df['컬럼명'] > 100 
이러면 이컬럼에 들어간 요소들중에 100이 넘는 요소들만 TRUE로 바꿔줘서 시리즈 형태가 하나 나와서
그거를 다시 



### 불린인덱싱으로 필터링한 df 
```python
df.loc[100보다큰거]
# 이런식으로 작성하면 100보다 작은거를 출력할수있다.
```
### 특정 컬럼을 보고싶은 필터링한 df
```python
# 컬럼 하나
df.loc[100보다큰거,'컬럼명']
# 여러 컬럼
df.loc[100보다큰거,['컬럼명','컬럼명']]

```
