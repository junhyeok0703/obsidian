

### 대표적으로 loc과 iloc이 있음

```python
burger_df.iloc[3,1]
# 인덱스(행) 인덱스3인거(4번째), 컬럼(열) 인덱스1(2번째인거)
burger_df.iloc[:,1]
# 다 가져오는데 컬럼은 인덱스 1번째인 2번째를 가져오는거
burger_df.loc[:['calories','carb']]
#이런식으로 loc은 이름을 가져와서 데이터를 가져올수있다. 

```

loc과 iloc을 사용해야지 행과 열을 정확하게 구분해놔서 뽑을때 좋은데
사용하지않고 
```
df['컬럼명':'컬럼명']
이런 로우 그니까 인덱스에 대한거는 이렇게 된다.
```
