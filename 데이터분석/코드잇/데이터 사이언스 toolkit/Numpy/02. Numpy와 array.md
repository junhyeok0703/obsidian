
### numpy 배열 만들기
- 넘파이 arr인 a라는 배열을 만든거다.
```
import numpy as np
a = np.array([1,2,3,4]);
```
### arange = 리스트의 range
```
# 0~9까지 들어있는 배열 만들었음
np.arange(10)
# 2~9까지 들어있는 배열 만들었음
np.arange(2,10)
# 2부터 9까지 2씩 증가하는 걸 만들었음
np.arange(2,10,2)
 -> 2,4,6,8
```
#### type을 보게 되면 numpy.ndarray
n차원 행렬이 만들어지게 되었다고 보여준다.
