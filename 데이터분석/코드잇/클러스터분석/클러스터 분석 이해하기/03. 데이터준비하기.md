### 이상치 제거
```python
import seaborn as sns

sns.set(style="darkgrid",
        rc={'figure.figsize':(16,9)})

# 데이터 시각화
sns.scatterplot(x=sales_df['total_buy_cnt'], y=sales_df['total_price'], s=200)

```
아주 큰 값을 가진 데이터가 하나 보이는데요. 이런 이상치는 결과에 영향을 줄 수 있기 때문에 제거해야 합니다.
